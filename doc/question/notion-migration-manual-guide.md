# 노션 독서 리스트 마이그레이션 수동 가이드

## 현재 상황
Notion MCP 도구의 `update-page` 기능에 기술적 제약이 있어 자동화된 마이그레이션이 어려운 상황입니다.

## 마이그레이션 작업 내용

### 목표
각 책 페이지의 본문에 있는 이미지와 텍스트를 "기록정보" 섹션으로 구조화하여 추가

### 작업 방식
각 책 페이지를 수동으로 편집하여 다음 형식으로 기록정보 섹션을 추가합니다:

```
## 기록정보

### 필사정보
[이미지 블록]

### 내생각정보
[이미지 다음에 오는 텍스트]

---

### 필사정보
[다음 이미지 블록]

### 내생각정보
[해당 이미지 다음에 오는 텍스트]

---
```

  1. 이미지 → "25.70.7 사랑에관하여"
  2. 이미지 → "257.8 부끄럽지않은 삶의 선택"
  3. 이미지 → "삶의 많은 고난에서...어둠이 아닌 빛을 바라본다"

### 주의사항
- 책읽는 이유는 기존 Properties의 "책읽는 이유" 필드에 그대로 유지
- 각 이미지-텍스트 쌍은 "---"로 구분
- 이미지 다음에 텍스트가 없으면 "내생각정보"는 비워두거나 생략 가능

## 자동화 가능 시점
Notion MCP 도구의 파라미터 형식 문제가 해결되면, 이 가이드를 기반으로 자동화 스크립트를 작성할 수 있습니다.

# 노션 독서 리스트 마이그레이션 계획

## 목표
노션의 "📖 독서 리스트" 데이터베이스에 있는 책들의 이미지와 텍스트 정보를 기록정보 형식으로 구조화하여 추가

## 작업 내용

### 1. 데이터 추출
- 각 책 페이지의 본문에서 이미지와 텍스트 블록 추출
- 이미지-텍스트 쌍으로 묶기 (이미지 다음에 오는 텍스트를 해당 이미지의 "내생각정보"로 연결)

### 2. 구조화
- 각 이미지-텍스트 쌍을 다음과 같이 구조화:
  ```
  ## 기록정보
  
  ### 필사정보
  [이미지]
  
  ### 내생각정보
  [텍스트]
  
  ---
  ```

### 3. 적용
- 각 책 페이지의 기존 본문 끝에 "기록정보" 섹션 추가
- 책읽는 이유는 기존 Properties의 "책읽는 이유" 필드에 그대로 유지

## 현재 상태
- Notion MCP 도구의 파라미터 형식 문제로 인해 작업 진행 중 어려움 발생
- `notion-update-page` 도구의 `data` 파라미터 형식 확인 필요

## 다음 단계
1. Notion MCP 도구의 정확한 파라미터 형식 확인
2. 각 책 페이지별로 이미지-텍스트 쌍 추출 및 구조화
3. 기록정보 섹션 추가

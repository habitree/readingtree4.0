## ⚠️ 발견된 누락 기능 (1개)

### 기록을 모임에 공유하는 UI 누락

**상태:**
- ✅ 백엔드 함수(`shareNoteToGroup`) 구현 완료
- ✅ Hook(`hooks/use-groups.ts`의 `handleShareNote`) 구현 완료
- ❌ 프론트엔드 UI 없음

**영향:**
- 사용자가 기록을 모임에 공유할 수 없음

**권장 수정 사항:**
1. 기록 상세 페이지(`app/(main)/notes/[id]/page.tsx`)에 "모임에 공유" 버튼 추가
2. 모임 선택 다이얼로그 구현
3. 선택한 모임에 기록 공유 기능 연결

## 📋 테스트 시나리오

### 시나리오 1-1: 카카오 로그인

**프롬프트:**
```
브라우저를 열고 https://readingtree2-0.vercel.app (또는 http://localhost:3000)에 접속하세요.

1. 로그인 페이지로 이동합니다 (/login)
2. "카카오로 로그인" 버튼을 클릭합니다
3. 카카오 로그인 화면에서 로그인을 완료합니다
4. 콜백 처리 후 리다이렉트되는 페이지를 확인합니다
5. 개발자 도구 콘솔에서 에러가 없는지 확인합니다
6. 네트워크 탭에서 OAuth 관련 요청이 성공적으로 완료되었는지 확인합니다

확인 사항:
- 로그인 버튼이 정상적으로 표시되는가?
- 카카오 로그인 화면으로 정상 리다이렉트되는가?
- 로그인 후 콜백이 정상 처리되는가?
- 에러 없이 메인 페이지 또는 온보딩 페이지로 이동하는가?
```

**예상 결과:**
- 카카오 로그인 화면으로 리다이렉트
- 로그인 성공 후 `/callback`으로 리다이렉트
- 온보딩 미완료 시 `/onboarding/goal`로 리다이렉트
- 온보딩 완료 시 메인 페이지(`/`)로 리다이렉트

### 시나리오 1-3: 온보딩 - 목표 설정

**프롬프트:**
```
온보딩 목표 설정 페이지(/onboarding/goal)에 있는 상태에서:

1. 페이지가 정상적으로 로드되는지 확인합니다
2. "올해 읽을 책 수" 입력 필드에 숫자를 입력합니다 (예: 12)
3. "시작하기" 버튼을 클릭합니다
4. 다음 페이지로 이동하는지 확인합니다
5. 개발자 도구 콘솔에서 에러가 없는지 확인합니다
6. 네트워크 탭에서 목표 설정 API 요청이 성공적으로 완료되었는지 확인합니다

확인 사항:
- 입력 필드가 정상적으로 작동하는가?
- 유효성 검사가 작동하는가? (1-100 범위)
- 목표 설정이 성공적으로 저장되는가?
- 다음 페이지로 정상 이동하는가?
```

**예상 결과:**
- 목표 설정 성공 토스트 메시지 표시
- `/onboarding/tutorial` 또는 메인 페이지로 이동

### 시나리오 1-4: 온보딩 - 튜토리얼

**프롬프트:**
```
온보딩 튜토리얼 페이지(/onboarding/tutorial)에 있는 상태에서:

1. 페이지가 정상적으로 로드되는지 확인합니다
2. 튜토리얼 내용을 확인합니다
3. "시작하기" 또는 "완료" 버튼을 클릭합니다
4. 메인 페이지로 이동하는지 확인합니다
5. 개발자 도구 콘솔에서 에러가 없는지 확인합니다

확인 사항:
- 튜토리얼 페이지가 정상적으로 표시되는가?
- 버튼이 정상적으로 작동하는가?
- 메인 페이지로 정상 이동하는가?
```

**예상 결과:**
- 튜토리얼 페이지 표시
- 메인 페이지(`/`)로 이동

### 시나리오 1-5: 세션 유지 확인

**프롬프트:**
```
로그인 완료 후:

1. 메인 페이지에서 헤더의 프로필 아이콘을 확인합니다
2. 프로필 메뉴가 정상적으로 표시되는지 확인합니다
3. 페이지를 새로고침합니다 (F5)
4. 로그인 상태가 유지되는지 확인합니다
5. 다른 탭에서 같은 사이트를 열어봅니다
6. 로그인 상태가 유지되는지 확인합니다
7. 브라우저를 닫았다가 다시 열어봅니다
8. 로그인 상태가 유지되는지 확인합니다

확인 사항:
- 새로고침 후에도 로그인 상태가 유지되는가?
- 다른 탭에서도 로그인 상태가 유지되는가?
- 브라우저 재시작 후에도 로그인 상태가 유지되는가?
```

**예상 결과:**
- 새로고침 후에도 로그인 상태 유지
- 다른 탭에서도 로그인 상태 유지
- 브라우저 재시작 후에도 로그인 상태 유지 (쿠키/세션 저장)

---

### 시나리오 1-6: 로그아웃

**프롬프트:**
```
로그인 상태에서:

1. 헤더의 프로필 아이콘을 클릭합니다
2. 드롭다운 메뉴에서 "로그아웃"을 클릭합니다
3. 로그인 페이지로 리다이렉트되는지 확인합니다
4. 개발자 도구 콘솔에서 에러가 없는지 확인합니다
5. 네트워크 탭에서 로그아웃 API 요청이 성공적으로 완료되었는지 확인합니다

확인 사항:
- 로그아웃 버튼이 정상적으로 작동하는가?
- 로그아웃 후 로그인 페이지로 이동하는가?
- 로그아웃 후 세션이 정상적으로 삭제되는가?
```

**예상 결과:**
- 로그아웃 성공 토스트 메시지 표시
- 로그인 페이지(`/login`)로 리다이렉트
- 세션 삭제 확인
